# ServerMonitor - Home Server Dashboard
# Data directory for storing metrics and logs

This directory contains historical data and logs for ServerMonitor.

## Contents

- `latest.json` - Latest metrics snapshot
- `metrics.json` - Metrics history (if using JSON storage)
- Log files - Application and error logs

## Notes

- Automatically created by the application
- Requires write permissions (755)
- Can be configured in `src/environment.php`
